## 🗺️ 1. Концептуальний Паспорт Сигналу (Signal Data Passport)

Кожне окреме повідомлення, знайдене скраперами Bright Data, перед збереженням в аналітичну базу 
зобов'язане пройти через 5 обчислювальних затворів. Це ліквідує хаос («білий шум») та автоматично 
визначає приховану дезінформацію.

### Фактори оцінки контенту та джерела:
1. **Релевантність (Relevance - R):** [0.0 ... 1.0] — Смислова відповідність ТЗ. Загороджувальний затвор.
2. **Актуальність (Recency - A):** [0.0 ... 1.0] — Часове згасання з урахуванням типу інформації.
3. **Значущість (Importance - V):** [0.1 ... 1.0] — Масштаб впливу на систему (від 1 до 10 балів).
4. **Сентимент (Sentiment - Snt):** [-1.0 ... +1.0] — Емоційний вектор (Негатив / Нейтрал / Позитив).
5. **Істинність (Truth - T):** [-2.0 ... +2.0] — Подвійний контур об'єктивності (Без нуля).

## ⏳ 2. Математика Актуальності: Ентропія та Параметр Статичності (S)

ШІ-агент розраховує часове знецінення інформації за динамічною формулою згасання:

$$A = \frac{1}{1 + (\text{Днів назад} \times S)}$$

Де $S$ — коефіцієнт деградації даних залежно від їхньої природи:

* **Клас 1: Абсолютна статика ($S = 0.0$).** Події, що завершилися. Історія, географія, факти смерті.
  * *Приклад:* "Іван помер", "Координати цеху №3".
  * *Результат:* $A = 1 / (1 + 0) = 1.0$ (Вічна актуальність, вага не падає з роками).
* **Клас 2: Трендова статика ($S = 0.2$).** Структурні параметри, вакансії, бюджети, контракти.
  * *Приклад:* "Завод відкрив 50 вакансій токарів", "Уряд затвердив оборонне замовлення".
  * *Результат:* Через 30 днів $A = 1 / (1 + 6) = 0.14$ (Зберігається як базова лінія Baseline).
* **Клас 3: Радикальна динаміка ($S = 1.0$).** Оперативна обстановка, новини, ціни, чутки в чатах.
  * *Приклад:* "Горить склад нафтобази Х", "Ціна акцій впала".
  * *Результат:* Через 3 дні $A = 1 / (1 + 3) = 0.25$ (Інформація знецінилася на 75%).

## 🕵️‍♂️ 3. Контур Об'єктивності: Позиціонування та Авторитет Джерела

Машина визначає істину ($T$) не суб'єктивно, а через математичне зіставлення емоцій повідомлення 
та прихованої фінансової чи політичної мотивації джерела.

1. **Авторитетність ($C_p$):** [1.0 ... 2.0] — Рівень верифікованої компетентності джерела в темі.
2. **Позиціонування / Упередженість ($B$):** [-1.0 ... +1.0] — Зацікавленість автора у викривленні.
   * `+1.0` — Кровно зацікавлений хвалити / приховувати проблеми (Представник компанії, прес-служба).
   * `-1.0` — Зацікавлений критикувати (Опозиція, конкурент, жертва інциденту).
   * `0.0` — Незалежний зовнішній аудитор (Сухі реєстри, митниця, супутникові знімки).

### Логіка Розрахунку Індексу Істинності ($T$):
* Якщо зацікавлене джерело ($B > 0.7$) видає аномально позитивний сентимент ($Snt > 0.7$) на фоні 
  інфраструктурної кризи — ШІ перемикає Індекс Істинності у **Негативний Дзеркальний Контур**:
  $$T = -1.5 \times C_p$$
  Вага сигналу зберігається, але вмикається алгоритм **інверсії сенсу** (читати текст навпаки).
* Якщо джерело нейтральне ($B = 0.0$), або його заяви підтверджуються іншими каскадами:
  $$T = 1.0 \times C_p$$

## 🧮 4. Підсумкова Аналітична Вага Сигналу (Signal Mass)

Фінальна вага об'єкта в базі знань, яка передається на Ланцюги Маркова та Графи знань, є добутком:

$$\text{Final Weight} = R \times A \times V \times |T|$$

* Якщо $R < 0.4$, то $\text{Final Weight} = 0.0$ (Сигнал миттєво відсіюється як сміття).
* Якщо $T < 0$, сигнал отримує повну системну вагу, але обробляється через вектор `INVERTED_MIRROR`.
